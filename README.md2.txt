** Problem Set 02 - Bank Marketing Prediction

** Objective
Predict whether a customer will subscribe to a term deposit using Logistic Regression.

** Dataset
Bank Marketing Dataset with 17 features.

** Methodology
1. Data loading using pandas
2. Data cleaning
3. Label Encoding
4. Train-test split
5. Feature scaling
6. Logistic Regression model training
7. Model evaluation

** Results
Achieved accuracy of around 88%

** Conclusion
Logistic Regression works well for binary classification problems like this.
